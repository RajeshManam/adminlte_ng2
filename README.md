# adminlte_ng2
angular2   adminlte 模板

工具:vscode

---
### 介绍
使用 angular2 改造 adminlte模板

### 编译
1. 通过git 克隆代码库:  git clone https://github.com/ZhaoRd/adminlte_ng2.git
2. 命令行切换到 src/adminlteng2
3. 执行 npm install -g angular-cli
4. 执行 npm install 
5. 执行 ng serve
6. 浏览器打开 http://localhost:4200
